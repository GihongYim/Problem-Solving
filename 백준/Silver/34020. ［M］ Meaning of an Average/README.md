### 문제 설명

<p>히바이는 최근에 만든 예술 작품을 광고하기 위해 별점을 조작하려고 한다. 물론 없는 별점을 막 만들었다가는 당연히 조작이 들통날 것이기 때문에, 별점의 평균만을 사용하여 작품의 별점을 조작하기로 했다.</p>

<ol>
	<li>우선 히바이는 자신의 작품에 매겨진 $N$개의 별점을 모두 담은 리스트를 만든다.</li>
	<li>이후, 리스트에 정확히 $1$개의 별점이 남을 때까지 아래 과정을 반복한다.
	<ul>
		<li>리스트에 있는 별점 중 $2$개 이상을 선택한다.</li>
		<li>선택한 별점을 리스트에서 삭제한 뒤, 선택한 별점의 평균을 리스트에 추가한다.</li>
	</ul>
	</li>
	<li>위 과정을 끝낸 뒤 최종적으로 리스트에 남아있는 별점이 작품의 별점이 된다.</li>
</ol>

<p>위와 같은 방식으로 작품의 별점을 조작할 때 히바이가 얻을 수 있는 작품의 별점의 최댓값을 구해 보자.</p>

### 입력 

 <p>첫째 줄에는 별점의 개수 $N$이 주어진다. $(1\le N\le 200\, 000)$</p>

<p>둘째 줄에는 $N$개의 별점을 의미하는 정수 $A_1,A_2,\ldots ,A_N$이 공백으로 구분되어 주어진다. $(0\le A_i\le 10^9)$</p>

### 출력 

 <p>첫째 줄에 히바이가 얻을 수 있는 작품의 별점의 최댓값을 출력한다.</p>

<p>실제 정답과 출력한 값의 절대오차 또는 상대오차가 $10^{-9}$ 이하이면 정답으로 처리된다.</p>
